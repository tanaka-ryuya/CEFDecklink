パイプラインロジック解説

A. プレロール同期キュー (時間制御)

CEFがフリーランで描画データをプッシュし、DeckLinkがポーリングでテクスチャを取り出します。
  • プレロール待機: キューが0から増えた時、3サイクル（約100ms）消費を一時停止してキューを「貯金」します。ジッターを吸収するためです。
  • 枯渇時 Freeze: キューが空になると、前回出力フレームを複製し、プレロール待機にリセットします。

B. Weave（織り交ぜ）インターレース合成

if (y % 2 == 0) { pixel = SampleFiltered1(x, y); // Frame 1 } else { pixel = SampleFiltered2(x, y); // Frame 2 }
縦のライン座標 y の偶数行を過去フレーム (F1)、奇数行を最新フレーム (F2) からマッピングします。動きがある箇所では、1枚の絵の中に前後のコマが混在するため「櫛状のブレ（インターレース縞）」が発生します。

C. 垂直 LPF（Anti-Flicker）

インターレース出力時に、隣接する走査線間の極端な色の差によって発生する「線フリッカー（チラつき）」を抑えるため、縦方向にローパスフィルタ（ぼかし）を適用します。
// 3-Tap Kernel: [0.25, 0.50, 0.25] pixel[y] = above*0.25 + center*0.50 + below*0.25; // 5-Tap Kernel: [0.0625, 0.25, 0.375, 0.25, 0.0625] pixel[y] = above2*0.0625 + above*0.25 + center*0.375 + ...

D. Unpremultiply（アルファ復元）

CEFの描画バッファは透明度を含んだ RGB がアルファ値で事前に乗算された Premultiplied Alpha 状態です。DeckLinkで「透明マスク（Key）」と「色信号（Fill）」を完全に切り分けるため、アルファ値で割り算を行い元の鮮やかなRGBカラーを復元します。
if (pixel.a > alphaThreshold) { pixel.rgb /= pixel.a; }
※アルファ値が小さすぎる（透明に近い）ときに割り算を行うと、ゼロ除算（または数値の暴走）が発生するため、alphaThreshold（閾値）を下回る場合は処理をスキップします。
